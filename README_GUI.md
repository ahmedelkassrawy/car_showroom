# Car Showroom GUI

This is a graphical user interface for the Car Showroom Management System.

## Requirements
- Python 3.x
- Tkinter (usually included with Python)

## How to Run

### Windows
Double-click `run_gui.bat` or run the following command in Command Prompt:
```
run_gui.bat
```

### Manual Run
Run the following command from the project root directory:
```
python car_showroom/gui.py
```

## Features

### Admin
- Login with `admin` / `admin123`
- Manage Cars (Add, Update, View, Delete)
- Manage Showrooms (Add, View)
- Manage Garages (Add, View)
- Manage Services (Add, View)
- View Customers
- View Reservations
- Manage Service Queue (View, Process Requests)
- View System Statistics

### Customer
- Register new account
- Login with existing account
- Browse Cars (Search by Make, Model, Price)
- Buy Cars
- Rent Cars
- View My Reservations (Cancel Reservation)
- Browse Garages & Services
- Book Services
- View Service Request Status
- View Transaction History
